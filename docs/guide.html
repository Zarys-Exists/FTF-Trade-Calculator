Introduction

This site has a variety of uses like calculating the value of your ftf trades, keeping track of your ftf collection and more. This guide covers most of the basic and some of the more subtle features of the site. You are welcome to send any feedback or suggestions to my discord (@Zvarys)

Trade Calculator

  • Value Source: All values displayed on this site are from the official FTF values website and are usually updated within 10 minutes of each value list update.
  • Items: The items are sorted by rarity and in the decreasing order of their value. Clicking on a filled slot will remove that item from the calculator. The "Adds" button allows you to add a hypothetical item that adopts the value of your choosing.
  • Set/Broken: Items are by default added to the calculator as a set and will have the set value as per the value list. This can be changed by clicking the 'H' or 'G' button before choosing the item, which will make the site consider it as a hammer or gem respectively and show the appropriate value.
  • Hammer/Gem ratio: The value of a broken (h/g) item is dependent on a few factors, including its rarity. For legendaries, the value of a set is split with 70% to the hammer and 30% to the gem. For seasonals, it's 50/50. However, there is an exception for sets with neon gems (aurora, cherry pie, etc.), in which case the ratio is instead 20/80 for hammer/gem respectively.
  • Unit Switching: You can switch between the old (HV) and new (FV) units of value through the toggle at the bottom right of the calculator. 1 HV = 40 FV
  • Item Stability: The border color of an item slot indicates the stability of that item, which relates to the current demand of that item. The colour code for each stability is given below.
    Rising Doing Well Improving Stable Fluctuating Struggling Receding Dropping

Inventory

  • Usage: You can use the Inventory feature to track your ftf collection, fuel your addiction to keep growing it, calculate your inventory value and more. Press "Add items" to start cataloguing your items.
  • Items: Unlike the trade calculator, you can choose multiple items at once to reduce the hassle. If you would like to add broken items, just press the h/g button for those specific items, and disable it while choosing sets. Pleaes note that pressing 'Cancel' will lead to the selected items being discarded. Items that have already been added will have an "Owned" badge on it, but you can still add it again.
  • Stats Stats are displayed for all sets present in your inventory. It shows you your total inventory value by rarity, completion rate for each rarity and quantity of items. Please note that broken items are NOT displayed in the completion stats nor the Unique/Total quantity, however, their value is accounted for in the total value. Also, the value stats do not show decimal values so if your net worth is a yeti hammer...whoops.
  • Sorting: Items are by default sorted by value but you can choose to sort by name, rarity or added order. You can also use the arrow button next to the sort to reverse the sorting order.
  • Save as Image: This button generates an image (or multiple) of your inventory. You can also write a custom note which will be shown on each image. A maximum of 56 items can be displayed per page, and the items are displayed in the same order they are sorted in. Each image shows the total value of every item in that image, their rarity and quantity. Similar to the trade calculator, the border of each item shows its stability.
